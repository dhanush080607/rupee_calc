# 💰 RupeeCalc

RupeeCalc is a free financial calculator website built using HTML, CSS, and JavaScript. It helps users make informed financial decisions through easy-to-use calculators and educational finance content.

## 🚀 Features

* 📈 SIP Calculator
* 🏦 EMI Calculator
* 💰 FD Calculator
* 🔄 Compound Interest Calculator
* 🧾 GST Calculator
* 📊 Simple Interest Calculator
* 🌙 Dark Mode Support
* 📱 Fully Responsive Design
* 📝 Finance Blog Section
* 🔍 SEO-Friendly Structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
* Vercel (Deployment)
